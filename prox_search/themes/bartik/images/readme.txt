/*****************************************************************************************************
Apple Touch Icons
*****************************************************************************************************/

For iOS devices (and Android, to a degree), you can create these files as the specified dimensions
(the last one should be 57px x 57px).  You can just include the large size and devices needing 
smaller sizes will scale it (at the expense of downloading a larger file) if they cannot find
their preferred size.  

apple-touch-icon-114x114.png
apple-touch-icon-72x72.png
apple-touch-icon.png

You can also use 'precomposed' versions that already have the drop shadow/glass effect added to 
them instead of relying on iOS to add that in and to have better Android compatibility. 
See http://mathiasbynens.be/notes/touch-icons for more information, and be sure to modify 
the image links in html.tpl.php if you use precomposed versions or if you want to remove
calls to sizes you do not create.
