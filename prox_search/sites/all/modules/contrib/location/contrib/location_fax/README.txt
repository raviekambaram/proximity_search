INTRODUCTION
------------

The Location Fax module, part of the Location package,
(https://www.drupal.org/project/location), allows users to to add a fax
number to a location.

* For a full description of the module visit
https://www.drupal.org/documentation/modules/location

* To submit bug reports and feature suggestions, or to track changes visit
https://www.drupal.org/project/issues/location


REQUIREMENTS
------------

This module requires the following modules:

* Location - (https://www.drupal.org/project/location)


INSTALLATION
------------

* Install the Location module as you would normally install a contributed
Drupal module. Visit https://www.drupal.org/node/895232 for further information.

CONFIGURATION
--------------

Navigate to Administration > Structure > Content Types >
[Content type to edit] > Locative information.
A fax number field is now available for configuration on node type
administration pages, the user settings page, CCK location field configure
pages and taxonomy edit pages.
