INTRODUCTION
------------

The Location CCK module, part of the Location
(https://www.drupal.org/project/location) package, defines a location field type.

* For a full description of the module visit
https://www.drupal.org/documentation/modules/location

* To submit bug reports and feature suggestions, or to track changes visit
https://www.drupal.org/project/issues/location


REQUIREMENTS
------------

This module requires the following modules:

* Location - (https://www.drupal.org/project/location)


INSTALLATION
------------

* Install the Location module as you would normally install a contributed Drupal
module. See
https://www.drupal.org/docs/7/extending-drupal-7/installing-contributed-modules-find-import-enable-configure-drupal-7
for further information.


CONFIGURATION
--------------

1. Navigate to the modules page and enable the Location CCK module.
2. Now location fields can be added in the same manner as other fields on the
"Manage fields" tab of a content type.
3. Collections Settings are now available including: Location name, Street
location, Additional, City, State/Province, Postal Code, Country and Coordinate
Chooser. Each setting has configuration and choice of default values.
4. Display settings to hide fields are also now available.
